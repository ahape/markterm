# Markterm

Show rendered markdown in your terminal.

## Setup

### Requirements

* Python runtime

### Configuration

For first-time installation, clone the repo, `cd` into it, and run `source
.venv/bin/activate` (bash-like shell) or `.venv/Scripts/activate` in Windows
Command Prompt or Powershell. 

Then, run `pip install -r requirements.txt`.

Now you're ready.

## Usage

Just run `Show-Markdown <filename>` (Windows Powershell) or `./show_rendered_markdown <filename>` (bash-like)
